## 说明

这个web应用，可以绕过appstore实现文件下载，绿色版，无需担心第三方bin存在后门等问题

## 安装

将 src 下面的文件，放到web目录下，然后用 `bower` 安装所需要的库

```
bower install angularjs bootstrap jquery angularLocalStorage angular-cookies
```
